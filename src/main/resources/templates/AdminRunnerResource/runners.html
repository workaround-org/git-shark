CI runners

Runners execute repository workflows. git-shark speaks the Forgejo/Gitea runner protocol, so a stock forgejo-runner or act_runner registers against this instance unchanged. Generate a registration token below, then run:

forgejo-runner register --no-interactive --instance <this instance URL> --token <registration token>
